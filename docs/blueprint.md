# **App Name**: GoAlong Community Mobility MVP

## Core Features:

- User Authentication: Allow users to register, login, and manage their profiles (dummy data).
- Route Planning (Driver): Enable drivers to input their regular commute route with start and end points, time of travel, and available seats.
- Route Matching: Match riders and parcel senders with available drivers based on route proximity and timing.
- Ride/Delivery Request (Rider/Sender): Allow riders/senders to request a ride/delivery by specifying their pickup/drop-off locations and desired time.  Includes the use of a generative AI 'tool' to assist riders/senders who do not know the locations exactly
- Cost Estimation: Calculate and display the estimated cost of the ride or delivery based on distance and a fixed rate.
- Booking Confirmation: Provide a confirmation screen with ride/delivery details after a match is confirmed (dummy data).
- User Feedback (Mock): Allow users to give a dummy rating to drivers, to encourage safe travels and fair prices.

## Style Guidelines:

- Primary color: Deep sky blue (#3399FF) to convey trust and efficiency, while nodding to the theme of travel.
- Background color: Very light cyan (#E5F5FF) for a clean and airy feel.
- Accent color: Soft green (#80FF33) to signal environmentally friendly solutions.
- Font pairing: 'Poppins' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text, for a modern and accessible feel.
- Use simple, recognizable icons for navigation and features. Consider using icons related to transportation and community.
- Prioritize a clean and intuitive layout for easy navigation. Use clear visual hierarchy to guide users through key actions.
- Incorporate subtle transitions and animations to provide feedback and enhance the user experience, e.g., hover effects, smooth transitions, count-up animations, and loading/success states.
